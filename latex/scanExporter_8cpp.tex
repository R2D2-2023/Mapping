renders lidar scans with raylib and applies rotation from magnetometer and can save this as png\textquotesingle{}s. 

Lidar type\+: ydlidar x2l magnetometer sensor read via serial by pressing space you can save 1 scan as png renders using raylib (gpu accelerated)
